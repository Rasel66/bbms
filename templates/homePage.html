Overview of Blood Groups

A+
A strong ally for A+ and AB+ blood
  1. Antigen: A
  2. Rh factor: Present
  3. Can Donate To: A+, AB+
  4. Can Receive From: A+, A-, O+, O-
A-
A rare helper for A and AB types
  1. Antigen: A
  2. Rh factor: Absent
  3. Can Donate To: A+, A-, AB+, AB-
  4. Can Receive From: A-, O-
B+
B type enriched with Rh positivity
  1. Antigen: B
  2. Rh factor: Present
  3. Can Donate To: B+, AB+
  4. Can Receive From: B+, B-, O+, O-
B-
A rare champion for B and AB groups
  1. Antigen: B
  2. Rh factor: Absent
  3. Can Donate To: B+, B-, AB+, AB-
  4. Can Receive From: B-, O-
AB+
The universal receiver
  1. Antigen: A and B
  2. Rh factor: Present
  3. Can Donate To: AB+ (Universal Plasma Donor)
  4. Can Receive From: All types (Universal Recipient)
AB-
The rarest of them all
  1. Antigen: A and B
  2. Rh factor: Absent
  3. Can Donate To: AB+, AB-
  4. Can Receive From: A-, B-, AB-, O-
O+
The lifeline for all positives
  1. Antigen: None
  2. Rh factor: Present
  3. Can Donate To: O+, A+, B+, AB+
  4. Can Receive From: O+, O-
O-
The universal donor
  1. Antigen: None
  2. Rh factor: Absent
  3. Can Donate To: All groups (Universal Donor)
  4. Can Receive From: O-
